METHOD AND SYSTEM FOR IDENTIFYING A MOLECULE OF ARTIFICIAL DEOXYRIBONUCLEIC ACID (DNA) NUCLEOBASE

Overview
This project presents a comprehensive investigation into the electrical recognition of eight artificial DNA nucleobases (xDNA and yDNA). By embedding these nucleobases within a nitrogen-terminated graphene nanogap junction, their fingerprint transmission and current readouts were calculated. These readouts serve as a database for training multiple ML models to achieve precise basecalling accuracy.
